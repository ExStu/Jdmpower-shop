# Jdmpower-shop
E-Commerce store for selling car parts and more 
### In development at the moment | Находится в разработке

## Stack:

Frontend:
- NextJs
- Redux-toolkit
- React query
- Axios
- React-hook-form
- React yandex-maps
- TailwindCss

Backend:
- NestJs
- PrismaORM
- Argon2
- Passport-jwt
- PostgreSQL
